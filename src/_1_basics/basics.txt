1. Creating Variables and Naming them (Bicycle.java)
    There are 4 type of variables:
    - Instance Variables (Non-Static Fields): Technically speaking, objects store their individual states in
"non-static fields", that is, fields declared without the static keyword.
    - Class Variables (Static Fields): A class variable is any field declared with the 'static' modifier; this tells
the compiler that there is exactly one copy of this variable in existence, regardless of how many times the class has
been instantiated.
    - Local Variables: Similar to how an object stores its state in fields, a method will often store its temporary
state in local variables.
    - Parameters: The important thing to remember is that parameters are always classified as "variables" not "fields".

2. Creating Primitive Type Variables (PrimitiveType.java)
    The Java programming language is statically-typed, which means that all variables must first be declared
before they can be used.
    A variable's data type determines the values it may contain, plus the operations that may be performed on it.

    Data Type	        Default Value (for fields)
    byte, short, int	    0
    long	                0L
    float	                0.0f
    double	                0.0d
    char	                \u0000
    String / Object	        null
    boolean	                false

    Local Variables must be assigned a value before they can be used.

    Literals (hằng số trong code)
    - Integer literals: mặc định là int, thêm hậu tố L để thành long. Có thể viết ở hệ decimal (10),
hexadecimal (16, prefix 0x), binary (2, prefix 0b).
    - Floating-point literals: mặc định là double, thêm hậu tố F để thành float. Có thể dùng ký hiệu khoa học
(ví dụ 1.234e2).
    - Character & String literals: dùng ' ' cho char, " " cho String. Hỗ trợ Unicode và escape sequences
(\n, \t, \\, …).
    - null literal: dùng cho biến tham chiếu (object).
    - class literal: ví dụ String.class.

    Underscore trong số (Java SE 7+)
    - Có thể dùng _ để phân tách nhóm chữ số cho dễ đọc
    - Không được đặt _ ở đầu/cuối số, cạnh dấu chấm thập phân, hoặc trước hậu tố (L, F).

3. Creating Arrays (Arrays.java)
    Array là đối tượng chứa một số lượng cố định các giá trị cùng kiểu dữ liệu.
    Độ dài của array được xác định khi tạo và không thể thay đổi sau đó.
    Các phần tử được truy cập bằng chỉ số (index), bắt đầu từ 0.

4. Using Var Type Identifier
    Xuất hiện từ Java SE 10.
    Cho phép khai báo biến cục bộ mà không cần chỉ rõ kiểu, trình biên dịch sẽ tự suy luận.
    Kiểu dữ liệu sau khi được suy luận thì không thể thay đổi.

    Hạn chế của var
    - Chỉ dùng cho biến cục bộ trong phương thức, constructor, hoặc initializer block.
    - Không dùng được cho:
        + Trường (fields) của lớp.
        + Tham số phương thức hoặc constructor.
    - Biến phải có giá trị khởi tạo ngay khi khai báo, vì null không có kiểu để suy luận.

5. Using Operators
    Toán tử (operators) là ký hiệu đặc biệt thực hiện phép toán trên 1, 2 hoặc 3 toán hạng và trả về kết quả.
    Thứ tự ưu tiên (precedence) quyết định toán tử nào được thực hiện trước trong biểu thức.

Operators	                Precedence
postfix	                    expr++ expr--
unary	                    ++expr --expr +expr -expr ~ !
multiplicative	            * / %
additive	                + -
shift	                    << >> >>>
relational	                < > <= >= instanceof
equality	                == !=
bitwise AND	                &
bitwise exclusive OR	    ^
bitwise inclusive OR	    |
logical AND	                &&
logical OR	                ||
ternary	                    ? :
assignment	                = += -= *= /= %= &= ^= |= <<= >>= >>>=

    Toán tử gán (Assignment Operator)
    - = dùng để gán giá trị cho biến.
    - Có thể kết hợp với toán tử số học để tạo gán phức hợp (compound assignment), ví dụ: x += 1.

    Toán tử số học (Arithmetic Operators)
    - Bao gồm: +, -, *, /, %.
    - + cũng dùng để nối chuỗi (String concatenation).

    Toán tử một ngôi (Unary Operators)
    - + (dương), - (âm), ++ (tăng 1), -- (giảm 1), ! (phủ định boolean).
    - Có dạng tiền tố (prefix) và hậu tố (postfix), khác nhau về giá trị trả về.

    Toán tử so sánh và bằng (Equality & Relational Operators)
    - ==, !=, <, >, <=, >=.
    - Dùng để so sánh giá trị giữa các biến.

    Toán tử điều kiện (Conditional Operators)
    - && (AND), || (OR) với cơ chế short-circuit.
    - Toán tử ba ngôi ?: là dạng rút gọn của if-else.

    Toán tử kiểm tra kiểu (instanceof)
    - Kiểm tra một đối tượng có phải là instance của một lớp hoặc interface cụ thể.

    Toán tử bit và dịch bit (Bitwise & Shift Operators)
    - ~ (NOT), & (AND), | (OR), ^ (XOR).
    - Dịch trái <<, dịch phải >>, dịch phải không dấu >>>.
